WXWUGRAPHS

Short installation guide:
  • Unpack wug.zip and copy directory "wxwugraphs" and file "wugraphs.php" or "wxwugraphs.php" (for Saratoga/Carterlake templates) to root of your website.
  • In directory wxwugraphs, make the directory "cache" writable for PHP (chmod 777)
  • Open http://www.yoursite.com/wxwugraphs/configurator.php in your browser (default password is "wugraphs") and set & save graph configuration. If you have problems with configurator page, you can edit file "WUG-settings.php", but don't use Windows Notepad
    If you upgrading from version v 0.7.9b and lower, you must clear WU graphs cache using configurator page and clear cache in your browser.
  • Open http://www.yoursite.com/wxwugraphs/WUG-test.php in your browser and check if everything is correct
  • Check graphs start page - http://www.yoursite.com/wxwugraphs.php or http://www.yoursite.com/wxwugraphs eventually http://www.yoursite.com/wugraphs.php
  • Done

Upgrade guide:
  • Backup your configuration - copy file settings.php (located in "wxugraphs" directory) from your website to your hard drive.
  • Unpack wug.zip and copy directory "wxwugraphs" and file "wugraphs.php" or "wxwugraphs.php" (for Saratoga/Carterlake templates) to root of your website.
  • Copy file settings.php from your harddrive back to your website (to "wxwugraphs" directory).
  • Done.
