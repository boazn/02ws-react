== Description ==

A quicklook into Lightbox Pop

    ★ Create lightbox, dialog box (message box) and quick popup box
    ★ Full control on popup content
    ★ Standard WordPress content editor
    ★ Display position and dimension control
    ★ Timeout based popup display
    ★ Display based on browsed number of pages
    ★ Css settings like color, z-index, overlay opacity, border etc
    ★ Shortcode for displaying popup in specific pages

This plugin allows you to create a lightbox, a dialog box and a quick popup box with custom contents. You can customize the popup display by configuring various settings such as  position settings (height, width, top,left), display logic settings (time delay after page load, number of pages to browse, lightbox repeat interval) and style settings(z-index, overlay opacity, color, border etc). You can use the plugin to display any type of contents such as special promotions, subscription forms, social media icons, feedback forms, video presentations and much more.

1. Extract `lightbox-pop.zip` to your `/wp-content/plugins/` directory.
2. In the admin panel under plugins activate Lightbox Pop.
3. You can configure the settings from XYZ Lightbox menu.
4. It should now be completely set up and functional.

== Frequently Asked Questions ==

= 1. What is a "lightbox" ? =

A lightbox is used to display any content inside a popup within a webpage, while rest of the page is greyed out and is not clickable. You may also call it greybox or thickbox. It is normally used to show subscription forms, contact forms, promotions, etc

= 2. What is a "dialog box" or "message box" ? =

A dialog box / message box is almost similar to lightbox, except that the content outside the box is not greyed out. It is normally used to show special notices, social media icons  etc.

= 3. What is a "quick box" or "quick popup" ? =

A quick box / quick popup box is almost similar to message box, except that the content is displayed when a user clicks on a statically positioned link at the sides of the screen. It is normally used to show feedback forms.

= 4. The Lightbox Pop plugin is not working properly =

Please check the wordpress version you are using. Make sure it meets the minimum version recommended by us. Make sure all files of the `lightbox pop` plugin uploaded to the folder `wp-content/plugins/`

= 5. Can I use this plugin with other popup plugins ? =

It may interfere with each other if there are 2 or more popup plugins. 

= 6. Can I add my own styles for the overlay ? =

Yes, you can change the style of the overlay from the settings page. If you need to use an image for the overlay, you have to use the premium version.

= 7. Can I set repeat interval in minutes ? =

Yes, you can set repeat interval in hours as well as minutes.

= 8. What kind of contents can be placed in a popup? =

You may  use the popup to display any type of contents such as special promotions, subscription forms, feedback forms, social media icons, video presentations and much more.

= 9. Can I display lightbox,dialog box or quick box in specific pages only ? =

Yes, you can do this by using the shortcode provided from settings page.

= 10. Can I create multiple lightboxes/dialog boxes/quick boxes for different pages ? =

You need to purchase the premium version for having this feature.
