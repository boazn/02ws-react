Cashie Commerce is the fastest way to start selling on your WordPress blog.

== Description ==

Cashie Commerce is an easy to use shopping cart that allows you to quickly start selling on your WordPress blog. Cashie Commerce hosts all of the code and functionality for you so there is no additional software to download, install, or maintain. Just use this handy plugin to complete a few easy steps and you're on your way to making money.

= Features: =

* No coding or coding knowledge is required
* Easily customize your store's design to match your WordPress theme
* Accept credit card or PayPal payments
* Add any number of tax rules
* Real-time shipping rates with USPS, FedEx, UPS, Canada Post, Australia Post
* Support for multiple currencies and languages
* Unlimited products and unlimited options with inventory control

For more information, go to <a href=\"http://cashiecommerce.com/wordpress?utm_campaign=plugin_admin&utm_medium=WordPress_Plugin&utm_source=WordPress\">http://cashiecommerce.com/wordpress</a>

== Installation ==

1.  Extract the zip file and just drop the cashie-commerce folder in the wp-content/plugins/ directory of your WordPress installation.
2.  Activate the Plugin from Plugins page.
3.  Click on the Cashie Commerce admin menu and you're ready to go!

== Frequently Asked Questions ==

= How much does Cashie Commerce cost? =

Cashie Commerce is free to try as long as you want. When you are ready to start processing real transactions, Cashie Commerce just costs 2.5% of your monthly transaction volume. You only pay money when you make money.

= Do I need a PayPal account to use Cashie Commerce? =

Yes, you will need to have a PayPal account in order to receive payments. Once your store is live, you will also need to pay for your Cashie Commerce transaction fees using PayPal. Don't worry, its easy to sign up for a PayPal account and you can get an account with no monthly or up-front charges.
